return type  
void = 함수 결과가 없음.
int = 함수의 결과가 int로 나옴....등


함수 return type 함수명 (매개변수 리스트)
             void main ( );
             int main (void);
main함수가 끝나면 끝남 다른 함수는 상관 없음.

return 함수는 하나에 변수 하나만 써야됨
함수는 종료하는 경로가 여러가지 함수가 종료하는 경로에 모두 return을 써 주어야 됨

숙제 예제 꼭 참고..!

마지막에 호출한 함수가 가장 먼저 끝남

다른 함수에서 만든 변수는 그 함수에서만 보임
함수가 사라지면 그 함수에 있던 변수도 사라짐

매개변수 : 다른 함수도 보이는 변수(공동변수)

  함수(매개변수) 으로 사용

재귀호출은 반복이 아니라 새로 만드는 것
